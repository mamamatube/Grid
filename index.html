グリッドコンテナーの指定

大枠の<div>にグリッドコンテナを指定し、grid-template-columns・rowsでグリッドアイテムの配置を指定

<div class="container">
<div class="item-1">1</div>
<div class="item-2">2</div>
<div class="item-3">3</div>
<div class="item-4">4</div>
<div class="item-5">5</div>
<div class="item-6">6</div>
</div>

.container {
display: grid;
grid-template-colmns: 100px 100px;
grid-template-rows: 100px 100px 100px;
}
.item-1 { background: #d66a6a; }
.item-2 { background: #e6a97e; }
.item-3 { background: #f2d479; }
.item-4 { background: #a7c97f; }
.item-5 { background: #7bb38e; }
.item-6 { background: #6dbec9; }

グリッドアイテムの並び替え
デフォルトは左上から右に進むが、grid-auto-flow:column;を指定すると、左上から下に進んで並ぶようになる

.container {
display: grid;
grid-template-colmns: repeat(3, 100px);
grid-template-rows: repeat(2, 100px);
grid-auto-flow:column;
}
.item-1 { background: #d66a6a; }
.item-2 { background: #e6a97e; }
.item-3 { background: #f2d479; }
.item-4 { background: #a7c97f; }
.item-5 { background: #7bb38e; }
.item-6 { background: #6dbec9; }

gapプロパティでアイテム同士の余白を指定
justify-content、align-ccontentで、領域内でのコンテナーの位置を指定する（start、center、endを用いる）
（plate-container: end end;(両方同じ値の場合はplate-container: end;でもOK)でまとめることも可能）

.container {
display: grid;
grid-template-colmns: 100px 100px 100px;
grid-template-rows: 100px 100px;
height:300px;
gap:8px;
justify-content: end;
align-content: end;
}

コンテナー内のアイテム位置を指定する場合、grid-column、grid-rowを用いて移動したいアイテムに場所を指定する。
指定したい範囲が複数要素で被った場合、表示させたい方を優先するためにz-indexが活用できる

.container {
display: grid;
grid-template-colmns: 100px 100px 100px;
grid-template-rows: 100px 100px;
text-align: center;
}
.item-1 { background: #d66a6a;
grid-column: 2 / 4;
grid-row: 2 / 3;
z-index:1; }
.item-2 { background: #e6a97e;
grid-column: 3 / 4;
grid-row: 1 / 3; }

場所を指定する際、spanを用いて、○マス分と指定することも可能。例）2から２マス分＝2 / span 2;

.item-1 { background: #d66a6a;
grid-column: 2 / span 2;
grid-row: 2 / 3;
z-index:1; }
.item-4 { background: #a7c97f;
grid-column: span 2; }

グリッドコンテナー内のアイテムに対し、コンテナーのレイアウト設定を機能させたい場合は、
アイテム自体もグリッドレイアウトにした上で、引き継ぎたいプロパティに対しsubgridを指定する。

以下は、ulに対し.containerのgrid-template-rowsを引き継いだことで、各行の黒丸位置が横並びで揃っている

.container {
grid-template-columns: 200px 200px;
grid-template-rows: repeat(4, auto);
display: grid; }

ul { grid-row: span 4;
display: grid;
grid-template-rows: subgrid; }
